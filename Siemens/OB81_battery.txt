проверка исправности буферной батареи в PLC S7-400

Когда возникает ошибка вызывается один раз блок OB81. когда сбрасывается ошибка тоже вызывается единожды OB81

блок OB81 - Power supply Fault

  OB81_FLT_ID(#TEMP1) -byte 1.0 в блоке ob81

пример кода:
L OB81_FLT_ID  //При возникновения ошибки батарейки здесь будет W#16#21
L W#16#21 // загрузить в аккамулятор значение для сравнения
==I      // оператор сравнения
S M 1250.0 // устанавливаем бит 


  OB81_EV_CLASS(#TEMP0)- при ошибки записалось W#16#39 при сбросе ошибке W#16#38
пример кода:   // при возникновении ошибки бит установится, а при исчезновении сбросится 
A(
L OB81_EV_CLASS
L W#16#38
)
A(
L OB81_FLT_ID  
L W#16#21 
)
R M 1250.0

A(
L OB81_EV_CLASS
L W#16#39
)
A(
L OB81_FLT_ID  
L W#16#21 
)
s M 1250.0
